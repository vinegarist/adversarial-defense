## 4. 关键发现

### 4.1 降低攻击强度成功 ✅

**原参数** (`top_k=9`):
- 遮蔽后准确率: **28.74%**
- 遮蔽过于激进，语义信息丢失严重

**新参数** (`top_k=5`):
- 遮蔽后准确率: **47.36%**
- 保留了更多语义信息，人眼仍可识别

**结论**: 将 `top_k` 从 9 降到 5 成功降低了攻击强度，符合会议要求。

---

### 4.2 PGD-AT 异常现象确认 ⚠️

**实验数据**:
```
标准模型:     遮蔽攻击准确率 47.36%
PGD-AT:       遮蔽攻击准确率 64.58%
差异:         +17.22% (PGD-AT 更高!)
```

**与论文预期对比**:
- **论文预期**: PGD-AT 应该比标准模型对遮蔽攻击更脆弱（准确率更低）
- **实验结果**: PGD-AT 比标准模型更鲁棒（准确率更高）

**可能原因**:
1. PGD训练增强了模型对局部扰动的鲁棒性，间接提升了对局部遮蔽的抵抗能力
2. MNIST数据集较简单，特征冗余度高，PGD训练使模型学会了更鲁棒的特征提取
3. 遮蔽攻击和PGD攻击在特征空间上存在一定相似性

**建议**:
- 需要进一步研究PGD-AT对遮蔽攻击防御机制
- 在不同数据集上验证该现象

---

### 4.3 Occlusion-AT 过拟合现象 🔍

**异常数据**:
```
Occlusion-AT 干净样本准确率: 95.68%
Occlusion-AT 遮蔽攻击准确率: 97.70%
攻击成功率: -2.02% (负数!)
```

**解释**:
- Occlusion-AT 专门针对遮蔽攻击进行了训练
- 模型学会了"忽略"遮蔽区域，只关注未被遮蔽的特征
- 遮蔽攻击反而帮助模型过滤了噪声，聚焦关键信息

**问题**:
- Occlusion-AT 对遮蔽攻击防御极强 (97.7%)
- 但对其他攻击（FGSM、PGD）防御极差（根据原notebook: FGSM 3.24%, PGD 0%）
- 存在严重的**过拟合**问题

---

### 4.4 Mix-AT 综合性能最佳 🏆

**性能指标**:
```
干净样本准确率: 98.46% (接近标准模型)
遮蔽攻击准确率: 90.18% (防御很好)
攻击成功率: 8.28% (综合平衡)
```

**优势**:
- 平衡了干净样本准确率和对抗鲁棒性
- 同时防御遮蔽攻击和PGD攻击
- 推荐作为最佳训练策略

## 7. 结论与建议

### 7.1 结论

1. ✅ **降低攻击强度成功**: `top_k=5` 比 `top_k=9` 保留更多语义信息
2. ⚠️ **PGD-AT 异常现象存在**: 与论文预期相反，需要进一步研究
3. 🔍 **Occlusion-AT 过拟合**: 对遮蔽攻击防御极强，但对其他攻击脆弱
4. 🏆 **Mix-AT 最佳**: 平衡了多种攻击的防御能力

### 7.2 后续工作建议

1. **研究PGD-AT防御机制**:
   - 可视化PGD-AT和标准模型的特征图差异
   - 分析PGD训练如何影响对遮蔽攻击的鲁棒性

2. **改进Mix-AT**:
   - 尝试不同的混合比例 (0.3, 0.5, 0.7)
   - 探索其他攻击组合的混合训练

3. **迁移到其他数据集**:
   - 在CIFAR-10上验证当前发现
   - 验证PGD-AT异常现象是否普遍存在

4. **理论分析**:
   - 从理论上解释为什么PGD-AT对遮蔽攻击有防御效果
   - 分析遮蔽攻击和PGD攻击在特征空间的关联性
